Description

Libsdi ecapsulates the Serial Attached SCSI Device Interface (SDI) for Linux. Storage management applications can then use the API provided by this library to support multiple vendors SAS controllers on both 2.4 and 2.6 Linux kernels.

--  Andrew Patterson
Features

  • API maps closely to the SDI specification (see sdi header and libsdi header).

Source for the libsdi project is stored in a subversion repository. The project can be checked out using the following command:

For Read-only access:

   svn checkout svn://svn.berlios.de/svnroot/repos/libsdi


For Read-Write access (project members):

   svn checkout svn+ssh://svn.berlios.de/svnroot/repos/libsdi
